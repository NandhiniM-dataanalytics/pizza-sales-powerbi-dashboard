# Pizza Sales Analysis Dashboard (Power BI)

## Project Overview
This project analyzes pizza sales data to identify key business insights such as revenue performance, customer ordering behavior, and popular pizza categories. An interactive dashboard was created using Power BI to help visualize sales trends and support data-driven decision making.

## Dataset
The dataset contains 48,000+ sales records including order date, order time, pizza category, pizza size, quantity sold, and pricing details.

## Objectives
The goal of this project is to analyze pizza sales data and answer key business questions such as:

• What is the daily trend for total orders?
• Which days of the week generate the highest number of orders?
• Which months show the highest sales performance?
• What percentage of total sales comes from each pizza category?
• What percentage of total sales comes from each pizza size?
• What are the top 5 best-selling pizzas based on revenue, quantity sold, and number of orders?
• What are the bottom 5 worst-selling pizzas based on revenue, quantity sold, and number of orders?
